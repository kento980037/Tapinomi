# タピのみ　 〜タピオカコレクション〜

## What is it ?

2019年は「東京タピオカランド」が原宿にオープンしたり、「タピる」「タピ活」などの言葉が流行るなど、タピオカブームがおきました。




素材自体を公開できないものを用いているためソースコードのみでの公開となります．
